# Product Update

This release is about trust, not sparkle.

Wetzler now gives the host agent a clear revision packet: the original Markdown, the writing profile, Vale findings, and a checklist that protects structure. The tool does not rewrite by itself. It hands the agent enough context to revise without losing headings, code fences, links, or useful evidence.

The result should feel plainspoken and alive. Keep the reader oriented first; let the personality arrive after the facts have somewhere to stand.
